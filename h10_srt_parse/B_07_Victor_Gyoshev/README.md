## Installation

Add this line to your application's Gemfile:

```ruby
gem 'srtparse_library'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install srtparse_library

## Usage
require('srtparser_library')
SRTParser.parse_file('path to file')
